## 购物车作业

* 使用说明

* 步骤一
```
[root@shuke learnpy]# unzip atm.zip
[root@shuke bin]# cd atm/bin
[root@shuke bin]# python start.py
```

* 步骤二
1.商品列表       ---不需要登陆
2.购物车         ---需要登陆
3.结算          ---需要登陆
4.账户管理中心    ---需要登陆

* 账户管理中心使用说
1. 账户信息     ---根据账户ID进行查询
2. 冻结账户     ---需要使用admin账户进行操作
3. 注册
4. 还款
5. 转账
6. 退出
